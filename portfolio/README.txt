Animated portfolio project

Files:
- index.html
- styles.css
- script.js

Features:
- Flying bird animation in hero section
- Mouse spark / fireworks effect using canvas
- Cursor glow following mouse
- Glassmorphism cards
- Tilt hover cards
- Scroll reveal animations
- Theme toggle
- Fully static site with HTML, CSS, JavaScript only

How to use:
1. Open index.html in browser
2. Replace name, role, skills, projects, and contact details
3. Optionally add your real image inside the hero card section
